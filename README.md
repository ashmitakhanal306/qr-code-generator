# 🔳 QR Code Generator

## 📌 Description
This is a simple and interactive QR Code Generator web application that allows users to generate QR codes instantly by entering any text or URL.

---

## 🚀 Live Demo
👉 https://ashmitakhanal306.github.io/qr-code-generator/

---

## ⚙️ Features
- 🔤 Generate QR code from any text or URL
- ⚡ Instant output
- 🎨 Clean and responsive UI
- 📱 Works on both mobile and desktop

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- QR Code API

## ▶️ How to Use
1. Enter text or URL in the input field  
2. Click on **Generate QR**  
3. QR code will be displayed instantly
